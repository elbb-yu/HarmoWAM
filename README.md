# HarmoWAM

Static project page for **HarmoWAM: Harmonizing Generalizable and Precise Manipulation via Adaptive World Action Models**. Plain HTML / CSS / vanilla JS, no build step, suitable for GitHub Pages.

## Local Preview

```bash
python3 -m http.server 8000
```

Open <http://localhost:8000>.

## Structure

- `index.html` - project page
- `static/css/style.css` - page styles
- `static/js/main.js` - BibTeX copy behavior
- `static/papers/harmowam.pdf` - provided PDF
